================  tablet.sh — chuleta  ================
cd /home/sebastian/Jira-and-confluence-/responsive-proxy

--- CONECTAR (una tablet = UN solo transporte) ---
  ./tablet.sh usb    [nombre]   cable permanente   -> runtime USB   (mas estable)
  ./tablet.sh qr     [nombre]   pairing por QR     -> runtime WiFi
  ./tablet.sh wifi   [nombre]   endpoint conocido  -> runtime WiFi
  ./tablet.sh use               menu numerado: elegir tablet / cambiar tipo
  ./tablet.sh                   reconecta TODAS segun su tipo guardado
  ./tablet.sh status            ver cada tablet y por donde esta

--- OPERAR ---
  ./tablet.sh cap    [nombre]   captura de pantalla -> /tmp/<tablet>-<fecha>.png
  ./tablet.sh url               elegis URL(s) (ej 1 3 4) + tablet(s) (ej 1 2 o 'todos') -> Chrome

--- DEBUG / CHROME ---
  ./tablet.sh clear             limpia cache+cookies+SERVICE WORKER de Chrome (uno/varios/'todos')
  ./tablet.sh stop              cierra Chrome (uno/varios/'todos')
  ./tablet.sh logs   [nombre]   logcat de Chromium en vivo (Ctrl+C salir)
  ./tablet.sh rec    [nombre]   graba la pantalla -> /tmp (Ctrl+C para parar, max 180s)

-------------------------------------------------------
PRIMERA VEZ
  USB : enchufa el cable  ->  ./tablet.sh usb
  QR  : tablet -> Depuracion inalambrica -> "Vincular con QR"  ->  ./tablet.sh qr

EN LA TABLET (Chrome): abri la MISMA url que en tu local
  http://localhost:5173/#/login
  http://localhost:5050

DESVINCULAR / EMPEZAR DE CERO
  borra la linea en devices.env  (o vacialo)  +  adb disconnect

-------------------------------------------------------
NOTAS
  - USB = cable puesto, el tunel viaja por el cable (no se cae por WiFi).
  - WiFi = no dejes el cable enchufado o el tunel se pega al USB (UsbFfs).
  - 'clear' es tu amigo: si la tablet muestra algo viejo/cacheado, casi siempre
    es el service worker -> ./tablet.sh clear y recargas.
  - El puerto 80 (Moodle) NO se puede reenviar por adb -> va por el proxy de Vite.
  - devices.env -> fuente de verdad: nombre | serial | ip:puerto | tipo(usb|wifi)
